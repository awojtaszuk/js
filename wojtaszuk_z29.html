Odpowiedzi

1. Określanie właściwości obiektu

Właściwość danego obiektu można określić na dwa sposoby.

Sposób 1 – notacja kropkowa:

auto.marka = "Opel Omega";
auto.rok = 1996;
auto.cena = 25000;

Sposób 2 – notacja nawiasowa:

auto["cena"] = 25000;

2. Odwoływanie się do pól i metod

Do właściwości i metod obiektu odwołujemy się przez nazwę obiektu oraz kropkę. 

console.log(samochod1.kolor);
samochod1.jedz();

3. Cechy charakterystyczne konstruktora

• jest wywoływany automatycznie podczas tworzenia obiektu
• nazywa się tak samo jak klasa (funkcja)
• nie zwraca żadnej wartości
• może przyjmować argumenty
• do jego wywołania używane jest słowo kluczowe new

4. Konstruowanie obiektu (3 pola + 1 metoda)

var osoba_1 = {
    nazwisko: "Kowalski",
    imie: "Jan",
    zawod: "piekarz",
    wyswietl: function () {
        document.write(this.nazwisko + " " + this.imie);
    }
}

5. Znaczenie słowa this

Słowo kluczowe this pozwala odwołać się do pola lub metody obiektu z wnętrza tego obiektu.

Pola i metody w obiekcie oddzielamy przecinkami.

6. Konstruktor w JavaScript

function klient(nazwisko_k, imie_k, zawod_k) {
    this.nazwisko = nazwisko_k;
    this.imie = imie_k;
    this.zawod = zawod_k;
    this.wypisz = function () {
        alert(this.nazwisko + " " + this.imie);
    }
}

Konstruktor jest funkcją pełniącą rolę klasy. Słowo this odnosi się do aktualnego obiektu.

7. Słowo kluczowe new

Słowo kluczowe new służy do tworzenia nowego obiektu na podstawie konstruktora. 

var osoba1 = new klient("Kowalski", "Jan", "kierowca");
var osoba2 = new klient("Nowak", "Anna", "sekretarka");

8. Właściwość prototype

Właściwość prototype służy do dodawania metod i właściwości do obiektów tworzonych przez konstruktor. 

function klient() {
    this.nazwisko = "Bielski";
    this.imie = "Paweł";
}

klient.prototype.pisz_dane = function () {
    document.write(this.nazwisko + " " + this.imie);
}

klient.prototype.zawod = "kierowca";

var osoba1 = new klient();
osoba1.pisz_dane();

Każdy nowo utworzony obiekt na podstawie konstruktora klient posiada dodane metody i właściwości z prototype.
